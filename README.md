## Description

This is a repository dedicated to my microservice architecture with kafka and postgreSQL.

## Installation

```bash
$ yarn install
```

## Running the PostgreSQL database and Kafka

```
$ docker-compose up -d
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
